# Dead Bells

## A GMTK 2025 Game Jam Submission

> *Awaken in a dark world with a ticking life, aiming to reach the top of a Clocktower to escape a loop. Dying sends you back to the bottom. Find hints in notes scattered throughout the world. Inspired by "Dead Cells."*

**Dead Bells** is a 2D pixel-art action platformer developed from scratch in 4 days for the [GMTK Game Jam 2025](https://itch.io/jam/gmtk-2025). It was created using Lua and the LÖVE2D framework.

**Disclaimer:** This game is an incomplete prototype developed under the time constraints of the game jam.

---

## Features

*   **Temporal Rewind Mechanic:** Strategically reverse time to undo mistakes, evade attacks, or set up devastating combos.
*   **Dynamic Platforming:** Navigate intricate levels filled with challenging jumps, climbable surfaces, and environmental puzzles.
*   **Engaging Combat System:** Face off against a variety of distinct enemies with unique attack patterns and behaviors.
*   **Pixel Art Aesthetics:** Immerse yourself in a charming retro-inspired pixel art world.
*   **Love2D Framework:** Built using the lightweight and powerful Love2D engine.

---

## How to Play

The objective is to navigate through challenging levels, defeat enemies, and utilize your time-rewind ability to overcome obstacles and reach the top of the Clocktower.

### Controls

*   **Move Left:** `A` or `Left Arrow`
*   **Move Right:** `D` or `Right Arrow`
*   **Jump:** `W` or `Up Arrow` or `Spacebar`
*   **Climb:** `W` or `Up Arrow` / `S` or `Down Arrow` (on climbable surfaces)
*   **Drop Through Platform:** Double-tap `S` or `Down Arrow`
*   **Attack:** `Left Mouse Button`
*   **Rewind Time:** `R`

### Installation & Running

To run this game, you'll need to have LÖVE2D installed on your system.

1.  **Download LÖVE2D:** If you don't have it already, download and install the appropriate version of LÖVE2D for your operating system from [love2d.org](https://love2d.org/).
2.  **Download the Game:** Download the latest version from the [itch.io page](https://elfoteo.itch.io/dead-bells) or clone this repository.
    ```bash
    git clone https://github.com/elfoteo/GMTK-2025
    cd GMTK-2025
    ```
3.  **Run the Game:**
    *   **Windows/macOS:** Drag the `GMTK-2025/main.lua` file onto the LÖVE2D executable.
    *   **Linux:** Open a terminal in the game's directory and run:
        ```bash
        love .
        ```
    *   **Play it online on itch.io:** [link](https://elfoteo.itch.io/dead-bells)
